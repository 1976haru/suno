# Suno 반자동 입력기 (suno-helper)

Suno Weaver Studio에서 만든 곡 데이터를 suno.com 작성 화면에 빠르게 채워 넣는
로컬 보조 도구입니다. **필드 채우기까지만 자동화하고, 곡 생성(Create) 버튼은
사람이 직접 클릭합니다.**

## 원칙 (반드시 지켜야 함)

1. **완전 자동화 아님.** 이 스크립트는 제목/스타일 프롬프트/가사/Exclude
   텍스트 필드만 채웁니다. Create/Generate 버튼을 스크립트가 누르는 코드는
   존재하지 않으며, 앞으로도 추가해서는 안 됩니다.
2. **보이는 브라우저만 사용.** headless 모드는 사용하지 않습니다
   (`headless: false` 고정). 브라우저는 Playwright의 persistent context로
   실행되어, 사용자가 이미 로그인한 프로필(`.profile/`)을 그대로 재사용합니다
   — 이 스크립트는 로그인 정보(아이디/비밀번호/쿠키)를 다루거나 저장하지
   않습니다. 첫 실행 시 사용자가 직접 브라우저 창에서 로그인하면 됩니다.
3. **곡 간 이동은 사람이 결정.** 다음 곡으로 넘어가려면 터미널에서 Enter를
   눌러야 합니다. 무인으로 여러 곡을 연속 처리하는 루프는 없습니다.

## 사용 전 확인 사항 — 셀렉터 검증 필요

`selectors.json`의 CSS 셀렉터는 Suno 웹사이트의 일반적인 UI 패턴을 바탕으로
작성한 **최선 추정치**이며, 실제 suno.com 페이지에서 이 도구로 직접 검증된
것이 아닙니다. Suno의 UI는 예고 없이 바뀔 수 있습니다. 처음 사용할 때는:

1. `npm run helper -- <파일 경로>`로 실행해 브라우저를 열고,
2. 한 곡만 테스트해서 제목/스타일/가사/Exclude 필드가 실제로 채워지는지 확인하세요.
3. 셀렉터가 맞지 않으면 (터미널에 "⚠ ... 필드를 찾지 못했습니다" 메시지가
   뜨면) 해당 값이 자동으로 클립보드에 복사되니 직접 붙여넣으면 되고, 이후
   브라우저 개발자 도구(F12) → 우클릭 → 검사로 실제 필드의 셀렉터를 확인해
   `selectors.json`을 수정하세요. 코드를 고칠 필요 없이 이 파일만 바꾸면
   됩니다.

## 처음 설정

```bash
cd suno-helper
npm install
npx playwright install chromium
```

## 실행

```bash
npm run helper -- ../songs-output.json
```

또는 앱에서 내보낸 JSON(`suno-pack.json`)도 그대로 됩니다:

```bash
npm run helper -- ../suno-pack.json
```

Windows에서는 `run-suno-helper.bat`을 더블클릭해도 됩니다 (경로를 물어봅니다).

한 세션당 기본 최대 20곡까지만 처리합니다 (보수적인 기본값). 늘리려면:

```bash
npm run helper -- ../songs-output.json --max=40
```

80곡 팩 전체를 한 세션에서 처리하려면:

```bash
npm run helper -- ../songs-output.json --max=80
```

기본값을 크게 늘릴수록 계정 제재 위험이 커진다는 점은 동일하게 적용됩니다 (아래
"안전 안내" 참고).

## 진행 방식

1. 브라우저 창이 열립니다 (처음이라면 로그인하세요).
2. 터미널에 첫 곡 정보가 표시되면 Enter를 눌러 그 곡의 제목/스타일/가사/
   Exclude를 채웁니다. (건너뛰려면 `s` + Enter)
3. 브라우저에서 채워진 내용을 확인·수정한 뒤, **직접** Create를 클릭하세요.
4. 터미널로 돌아와 Enter를 누르면 다음 곡으로 넘어갑니다.
5. 모든 곡을 마치면 스크립트는 종료되지만 브라우저는 열려 있는 채로
   남습니다.

## 안전 안내

- 이 도구는 **본인 Suno 계정**에서, **사람이 매 단계를 확인하며** 사용하는
  보조 도구입니다. 완전 무인 자동화가 아닙니다.
- Suno의 서비스 약관상 자동화된 사용은 제한되거나 금지될 수 있습니다.
  이 도구를 사용해서 발생하는 계정 관련 문제(제재 등)의 책임은 전적으로
  사용자에게 있습니다.
- 대량·장시간·무인 사용은 계정 제재 위험을 높이므로 하지 마세요. 기본
  세션당 최대 곡 수(20)를 크게 늘리지 않는 것을 권장합니다.
- 매 곡의 진행이 사람의 Enter 입력에 의존하므로, 이 도구 자체에는 곡 간
  대기시간 설정이 없습니다(사람 페이스가 이미 자연스러운 속도 제한 역할을
  합니다).
